## Instrukcja pracy z repozytorium

### Klonowanie repozytorium
```bash
git clone git@github.com:bogdanmarysia/new_service_PD5371.git
```

### Przełączanie gałęzi
```bash
git checkout marketing
```

### Powrót do wcześniejszej wersji
```bash
git checkout <hash_commita>
```
